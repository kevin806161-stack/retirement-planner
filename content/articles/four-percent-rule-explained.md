# 4% 法則是什麼？退休金提領率新手指南

如果你開始研究退休規劃，很快就會碰到「4% 法則」這個名詞。這是退休理財圈最廣為流傳的提領率原則，許多 FIRE（財務自由）追求者也常用這套公式來反推自己需要存多少錢。這篇文章帶你搞懂它的原理、適用情境，以及限制。

## 4% 法則的起源

4% 法則最早來自 1994 年美國財務顧問 William Bengen 的研究，他分析過去美國股債市場數據後發現：如果退休時每年從投資組合中提領 4%（並隨通膨調整），在大多數歷史情境下，這筆退休金可以撐超過 30 年而不會耗盡。

這個結論後來被廣泛簡化成一個好記的原則：**只要你的退休金總額是「年支出的 25 倍」，理論上就能支撐長期退休生活**。

## 4% 法則的計算公式

公式本身很簡單，分成兩個方向使用：

**用提領率反推所需退休金總額**

```
所需退休金總額 = 年支出 ÷ 4%
（等於年支出 × 25）
```

舉例來說，如果你預計退休後每年花費 60 萬元：

```
60 萬 ÷ 4% = 1,500 萬元
```

也就是說，你需要準備 1,500 萬元的投資組合，理論上就能用 4% 提領率支撐長期退休生活。

**用退休金總額反推每年可提領金額**

```
每年可提領金額 = 退休金總額 × 4%
```

如果你已經存到 1,200 萬，每年可提領金額大約是：

```
1,200 萬 × 4% = 48 萬元（約每月 4 萬元）
```

## 4% 法則的運作邏輯

這套法則的前提，是退休金以一定比例配置在股票與債券（原始研究大約是 50% 股票、50% 債券），透過投資組合的長期成長，去抵銷每年提領造成的本金減少，並對抗通膨侵蝕。

也就是說，4% 法則並不是要你把錢存銀行不動，而是假設這筆退休金持續投資在市場中，靠複利成長來支撐長期的提領需求。

## 4% 法則適合台灣嗎？這些限制要注意

雖然 4% 法則簡單好用，但直接套用在台灣情境時，有幾個需要留意的限制：

**研究基礎是美國市場數據**

原始研究基於美國股市過去近百年的歷史報酬率，台灣股市的波動性、長期報酬率與美股不完全相同，直接套用可能產生偏差。

**沒有考慮退休後的醫療支出增加**

研究主要針對一般生活開銷，但退休後期（70、80 歲以後）醫療支出通常會明顯上升，這部分需要額外預留緩衝。

**低利率環境下的爭議**

近年有不少財經學者提出質疑，認為在持續的低利率環境下，4% 可能過於樂觀，部分保守派建議改用 3% 到 3.5% 作為更安全的提領率。

**沒有考慮勞保勞退的補充**

對台灣勞工而言，退休金來源不只有個人投資組合，還有勞保年金和勞退專戶這兩層保障，因此實際需要靠 4% 法則計算的「個人投資缺口」，會比單純套公式得出的數字來得小。

## 更保守的替代算法：3.5% 法則

如果你想要更穩健的安全邊際，可以改用 3.5% 提領率計算：

```
所需退休金總額 = 年支出 ÷ 3.5%
（等於年支出 × 約 28.6 倍）
```

同樣以年支出 60 萬元為例：

```
60 萬 ÷ 3.5% = 約 1,714 萬元
```

會比 4% 法則算出的目標金額更高，但相對也更能因應市場波動與長壽風險。

## 如何把 4% 法則應用在你的退休規劃中

實務上，建議把 4% 法則當作「快速估算工具」而非絕對準則，搭配以下做法會更穩健：

1. 用 4% 法則先抓出粗略的退休金目標範圍
2. 扣除預估的勞保、勞退給付，計算出個人投資組合需要補足的缺口
3. 視個人風險承受度，決定要採用 4% 或更保守的 3.5%
4. 定期（建議每 1-2 年）重新檢視試算結果，根據實際投資績效與生活開銷調整

## 結語

4% 法則的價值在於提供一個簡單、好記的起點，讓你能快速抓出退休金的大致目標，而不是要你盲目照單全收。搭配台灣本地的勞保勞退制度，以及自己實際的風險承受能力去調整提領率，才能讓退休規劃更貼近現實。
